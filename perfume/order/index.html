How it works

Four clean steps from selection to shipment.

The process is built to remove friction. It stays conversational and premium without forcing the user through a heavy checkout experience.

Step 01

Select the scent

Use Collections for a quick browse or Notes for a more precise fragrance-family decision.

Step 02

Specify size and concentration

Add the bottle format, desired concentration, and whether the order is for personal use or gifting.

Step 03

Confirm delivery details

Include the shipping name, address, preferred delivery window, and any special instructions.

Step 04

Receive the final confirmation

The concierge replies with the final summary, payment details, and shipment timing.

Bottle sizes

Choose the right scale for the way you wear fragrance.

The standard bottle size supports daily use, while the smaller format works well for travel, testing, or gifting. Both keep the same premium presentation.

Concentration

Balanced projection without losing clarity.

The perfume is formulated for a strong opening, a readable heart, and a base that remains clear rather than muddying into excess sweetness.

Gift packaging

Finished with the restraint of a luxury object.

Add a short note, request wrap-ready packaging, or ask for a discovery set if the recipient is still choosing a signature scent.

Shipping

Concierge-led confirmation before the order leaves.

Once the request is approved, the final details are confirmed by email so the bottle ships to the correct address with no ambiguity.

Gift-ready presentation

Wrapped like a finished object.

Every order can be elevated into a gift or discovery moment. That includes note cards, wrap requests, and the option to start with a sample set before committing to a full bottle.
